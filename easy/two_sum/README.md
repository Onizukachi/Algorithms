# Two Sum

- Ссылка: https://leetcode.com/problems/two-sum/description/
- Платформа: LeetCode
- Сложность: Easy

## Вводные
Дан массив целых чисел `nums` и целое число `target`.
Нужно вернуть индексы двух чисел, сумма которых равна `target`.

Гарантируется, что существует ровно одно корректное решение.
Один и тот же элемент нельзя использовать дважды.
Ответ можно вернуть в любом порядке.

## Что ожидается по условию
- Вход:
  - `nums` — массив целых чисел
  - `target` — целое число
- Выход:
  - массив из двух индексов элементов `nums`, которые в сумме дают `target`

## Ограничения
- `2 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`
- Существует только один валидный ответ

## Примеры
1. Input: `nums = [2,7,11,15], target = 9`  
   Output: `[0,1]`

2. Input: `nums = [3,2,4], target = 6`  
   Output: `[1,2]`

3. Input: `nums = [3,3], target = 6`  
   Output: `[0,1]`
