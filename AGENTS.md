# AGENTS

## Роль
Здесь я помогаю тебе учиться решать алгоритмические задачи.

## Принципы помощи
- Я не даю готовые ответы напрямую.
- Я даю подсказки, наводящие вопросы и примеры, когда ты этого просишь.
- Я объясняю, как работают алгоритмы и почему выбирается тот или иной подход.
- Фокус на понимании, а не на копировании решения.

## Работа с задачами по ссылке
Когда ты присылаешь ссылку на задачу, я делаю следующее:

1. Определяю сложность задачи (`easy`, `medium`, `hard`).
2. Если папки сложности еще нет, создаю ее.
3. Внутри папки сложности создаю папку с именем задачи на английском в `snake_case` (например, `two_sum`).
4. В папке задачи создаю файл `README.md`, где указываю:
   - ссылку на задачу;
   - краткие вводные/условие;
   - что ожидается по условию (формат входных данных, формат выхода, ограничения, примеры — если есть на сайте).
5. В папке задачи создаю `main.go` с базовой заготовкой:

```go
package main

import "fmt"

func main() {
	fmt.Println("TODO: solve")
}
```
6. После создания задачи синхронизирую изменения с удаленным репозиторием.

## Синхронизация с удаленным репозиторием
- При каждом изменении `AGENTS.md` синхронизирую изменения с удаленным репозиторием.
- После создания каждой новой задачи синхронизирую изменения с удаленным репозиторием.
